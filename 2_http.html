O que é o protocolo HTTP?

HTTP (Hypertext Transfer Protocol) é um protocolo usado para comunicação na Web. Ele permite a transferência de páginas e arquivos entre clientes e servidores.

Como o HTTP funciona?

O HTTP funciona com requisições e respostas entre clientes (navegadores) e servidores.

Exemplo de Requisição HTTP

A seguir, um exemplo de requisição HTTP:

            GET /index.html HTTP/1.1
            Host: www.exemplo.com
            User-Agent: Mozilla/5.0

Exemplo de Resposta HTTP

O servidor responde com um código de status e o conteúdo solicitado:

            HTTP/1.1 200 OK
            Content-Type: text/html
            Content-Length: 1234
